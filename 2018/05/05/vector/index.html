Vector

#介紹
可自行擴增的動態陣列
記得要引入<vector>標頭檔哦

#用法們

##建立
有幾種建立方法

  1. vector<type> name 開一個名為name的新陣列，裡面裝的是type這個型別
  2. vector<type> name(a, b)
    開一個名為name的新陣列，裡面裝的是type這個型別，然後預先塞進a個b（b要屬於type）

##增加

  1. name.push_back(a)
    在name的尾端放進一個a
  2. name.assign(a, b)
    把name裡面的元素全部清空（但不清掉已經佔用的記憶體）然後放進a個b
  3. name.reserve(a)
    把name所佔用的記憶體大小開到a。如果a小於原本的記憶體大小，不進行任何操作。
  4. name.insert(a, b)
    在a所指向的元素前面插入b（注意：a是指標！！）

##減少

  1. name.pop_back()
    把name最尾端的一個元素移除
  2. name.clear()
    把name清空（但不清掉已經佔用的記憶體）
  3. name.resize(a)
    把name的大小減到剩下a（但不清掉已經佔用的記憶體）
  4. name.erase(a)
    把a所指向的元素移除（注意：a是指標哦）

##存取

  1. name[a]
    取得索引值為a的元素的索引
  2. name.front()
    取得最前面的元素的索引（相當於name[0]）
  3. name.back()
    取得最後面的元素的索引（相當於name[name的大小-1]）
  4. name.at()
    跟name[]一樣，差別在於name.at()比較安全，不會越界存取，但也比較慢。

##迭代

  1. name.begin()
    取得最前面元素的指標
  2. name.end()
    取得最後面元素的指標

##取得參數

  1. name.size()
    取得name元素的總數
  2. name.capacity()
    取得name所佔的記憶體大小
  3. name.empty()
    如果name是空的就回傳True
    要不然就回傳False

##特殊用法
vector<bool> name
這個型別的vector有特別優化過，每個元素只佔1bit
